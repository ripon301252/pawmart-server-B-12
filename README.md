## ⚙️ Server Side (Backend) Overview
# 📍 Project Name: PawMart Server

## 🌐 Live Server Link:
### 👉 https://pawmart-server-seven.vercel.app/

# 🔹 5 Key Features:

## 🌐 Express.js REST API:
- A RESTful API created with Express.js to handle all client requests efficiently.

## 🧩 MongoDB Database Integration:
- All user, pet, and order data are securely stored and managed in MongoDB.

## 🔐 JWT Authentication:
- Ensures secure access with JSON Web Token (JWT)-based authentication and protected routes.

## ⚡ Complete CRUD Operations:
- Fully supports Create, Read, Update, and Delete functionalities for managing data.

## 🚀 Vercel Deployment:
- The backend is deployed on Vercel for reliable hosting and fast response times.
